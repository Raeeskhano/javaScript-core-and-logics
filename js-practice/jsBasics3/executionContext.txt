//*******************Js Execution Context*********************

in javascript, execution Context decides how the code will run.
✅ global execution Context --> allocate in (this) keyword.
✅ function execution Context
✅ eval execution Context

it run the code in two phases;
1- memory creation phase
2- execution phase 

ForExample;

let val1 = 10;
let val2 = 5;

function addNum(num1, num2){
    let total = num1 + num2;
    return total;
}

let result1 = addNum(val1, val2);
let result2 = addNum(10, 2);
